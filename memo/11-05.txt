지금까지의 연산은 숫자or문자끼리의 연산
논리연산자는 참과 거짓으로만 연산하는 연산자

&& : AND
T && T -> T
앞이 참이고 뒤가 참이면 연산결과가 참
T && F -> F
F && T -> F
F && F -> F
|| : or
앞,뒤 중 최소 하나라도 참이면 연산결과가 참
T || T -> T
T || F -> T
F || T -> T
F || F -> F

not 연산자
참 -> 거짓
거짓 -> 참
!T -> F
!F -> T

참과 거짓으로 하는 연산(중요)

let x = 10;

x>5 && x<20 -> x값이 5보다 크고 20보다 작냐?

5 < x < 20

2*(3+4)

let result = 값1 ?? 값2;

값1이 null거나 undefined일 때 값2를 반환
