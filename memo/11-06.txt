1. git을 우선적으로 깔아야 함
git -> 버전을 관리해주는 도구
우리 컴퓨터 작업한 내용

레파지토리(repository) -> 코드들을 저장하는 폴더
add -> 커밋하기전에 내가 작업한 내용을 임시공간에 올리는 명령어
commit -> 내가 지금까지 작업한 내용을 버전으로 만들어서 저장
여기까지는 내 컴퓨터 내부에서 하는 작업
push -> 커밋된 내용을 github로 올려주세요
clone -> github에 있는 특정 폴더를 내 컴퓨터에 복사하겠다.
ㄴ 레파지토리를 고르고 -> 내 컴퓨터 어디에 저장할지 경로를 설정 -> clone
