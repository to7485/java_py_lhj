자바의 자료형

기본자료형
- int, char, boolean, ...
- 값들을 직접 저장

참조자료형
String, 배열, 우리가 만든 클래스...
- heap 영역에 객체가 만들어지고 객체까지 접근할 수 있는 주소를 변수에 저장한다.

String 클래스
- 문자열은 객체(Object) 취급을 한다.
- 변수에는 주소가 저장되는게 맞다.
- String클래스의 toString()메서드가 문자열이 그대로 출력되도록 해준다.

indexOf("문자열") : 해당 문자열의 인덱스 값을 반환
charAt(index) : 문자열에서 해당 인덱스에 해당하는 '문자'를 반환
split("구분자") : 문자열을 구분자로 잘라서 배열로 만들어서 반환
length() : 문자열의 길이를 반환
euqals() : 두개의 문자열의 값이 같은지 비교(완전히 같아야 true)
equalsIgnoreCase() : 대소문자 가리지 않고 같은지 비교(Hello, hello, hEllo 다 true)
substring(offset, endIndex) : 시작 인덱스부터, endIndex 전까지 자른다.
replace("기존문자열","교체문자열") : 문자열을 교체해준다.

배열
- 같은 타입의 데이터를 여러개 저장할 수 있는 자료구조
- 배열에 들어있는 데이터를 요소(element)라고 부른다.
- 각 요소들의 위치번호를 index라고 한다.
- index는 0부터 시작하여 1씩 증가하면서 순차적으로 부여

배열의 생성
배열의 선언
- 자료형[] 변수명;

1. 초기화(initialization) 배열을 사용하는 방법
ex) {1,2,3,4}

int[] arr = {1,2,3,4}; -> jvm이 공간 4개짜리 배열을 메모리에 만들어준다.

2. 생성자를 이용하는 방법
ex) new Array(길이)

int[] arr = new Array(3); -> 공간 3개짜리 배열이 메모리에 올라가게 된다.
※대신 배열에 기본값을 넣고 시작한다.

배열에 있는 요소의 접근
배열명[index] -> 변수처럼 사용하자!!!

변수를 이용해서 출력도 가능하고 새로운 값을 대입하는것도 가능하다.

int [ ] arr = {1,2,3}

System.out.println(arr[0]) -> 1

arr[0] = 100; -> {100,2,3}

객체를 생성하기 위해 정의한 클래스
ㄴ 우리가 데이터타입을 만드는 것
자바로 사람에 대해서 정의할거야, 차에 대해서 정의할거야

public class Person{ -> 클래스(설계도)
    상태 -> 객체마다 가지는 값 (js에서는 property라고 불렀다.)
    이름;
    나이;
    키;
    몸무게;
}

객체(클래스를 통해서 사람을 만든다)
Person p = new Person();
p.이름 = 홍길동
p.나이 = 30;
p.키 = 159.6;
p.몸무게 = 48.7;

생성자
객체를 생성해주는 특수한 기능을 가지고 있는 메서드
따로 정의하지 않으면 컴파일러가 알아서 만들어줌 -> 우리 눈엔 안보임

생성자 정의하는 법

접근제한자 클래스명(){

}

생성자를 통해 값을 받아서 필드를 초기화 할 수 있다.

접근제한자 클래스명(매개변수){
    this.필드 = 매개변수;
}

기본생성자를 따로 정의하지 않고 매개변수가 있는 생성자를 작성하게 되면 기본생성자는
사용할 수 없다.
