컴퓨터에는 메모리가 있다(RAM) 
-> 자바 코드를 작성해서 실행하면 일정량을 빌려온다.
-> Stack과 Heap으로 구역을 나눈다.

Stack에 저장되는것					Heap 영역에 저장되는 것
지역변수 -> main 안에 작성되는 변수들			배열 [1,2,3]
매개변수 -> 함수의 소괄호 안에 있는 것			객체 { name : '홍길동',age:25}
객체변수 -> 객체까지 접근할 수 있는 주소를 저장


자바에서 문자열은 객체 취급을 한다.
객체는 Heap영역에 만들어진다.
변수는 객체가 저장된 장소까지 가는 주소값을 알고 있다.

대표적인 참조형 타입
String

같은 내용의 문자열이 있다면 서로 다른 변수라도 같은 주소를 갖는다.

암시적 객체 생성
String str = "홍길동";

명시적 객체 생성
String str = new String("홍길동");
ㄴ 새로운 객체를 생성하기 때문에 주소를 공유하지 않는다.

메서드
클래스 안에 정의된 함수

메서드를 사용하는 방법
1. 객체를 생성한다.
2. 생성된 객체 뒤에 점(.)을 찍고 메서드 이름을 작성한다.
String text = "hello";
int len = text.length() //length()메서드 호출
