연산자
연산을 도와주는 기능이 있는 기호
산술연산 : 사칙연산
비교연산 : 두 데이터 중 크고 작음을 비교
ㄴ 중요 : 비교연산 후 나온 데이터의 타입!!!
증감연산자 : 전위연산 ,후위연산 -> 헷갈리는 문제가 많이 나옴
논리연산자 : 논리형 데이터를 가지고 연산
ㄴ A && B : 그리고(and) 점수 : 0 미만은 안되고, 100 초과도 안됨
ㄴ A || B : 또는(or) : 배수 구할 때 3의 배수이거나 또는 5의 배수인경우
ㄴ !A : true -> false, false -> true
