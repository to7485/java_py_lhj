CSS를 주기 위해서 태그를 선택
클래스 값을 가진 태그를 선택하는 방법

클래스값을 가지고 있는 태그를 선택
.클래스값{
    스타일;
}

하나의 태그는 여러개의 클래스를 가질 수 있다.
클래스 값들은 공백을 기준으로 여러개를 작성할 수 있다.
<p class="class1 class2 class3">내용</p>

class 1,2,3라는 값을 모두 가진 p태그를 선택하려고 할 때
.class1.class2.class3{
	스타일
}

API(Application Programming Interface)

데이터를 주고받을 수 있도록 하는 규칙과 정의

자바의 변수
내가 변수에 넣으려는 데이터의 타입을 변수 앞에 작성을 해야 했다.
정수, 실수, 문자, 문자열, 논리형

파이썬의 변수
내가 변수에 넣는 값의 타입에 따라 자동으로 결정됨(동적 타입)

자바스크립트의 변수

변수명을 작성하는 규칙
1. 영문자, 숫자, _ $만 사용이 가능하다.
2. 숫자로 시작할수는 없다.
3. 대소문자는 구별한다 (User와 user를 다르다고 본다)
4. JS문법에 쓰이는 단어는 사용할 수 없다.
5. 한글 되긴하는데 그냥 없다고 생각하는게 좋다.
