HTML에 CSS를 적용하는 방식
1. 태그 안에 style속성으로 주기
2. <style>태그 안에 정의하기
3. 외부 파일에 작성하고 <link>태그로 불러오기

태그를 선택하는 방법
1. 태그를 직접 선택 (해당 페이지의 같은 태그가 전부 선택)
2. 클래스 선택자(태그에 class 속성을 준다.)
ㄴ .클래스명{스타일}
3. 아이디 선택자(아이디값이 겹치지 않게 정한다.)
ㄴ #아이디값{스타일}
4. 속성선택자(특정 속성 또는 특정 속성값을 가지는 태그를 선택)
ㄴ [속성] , [속성=값]

테두리 속성
border-width : 테두리 두께
border-style : 테두리 스타일
border-color : 테두리 색깔
border : 두께, 스타일, 색깔을 한 번에 지정할 수 있는 축약속성
border-radius : 테두리 둥글게 만듦

배경(background)스타일
background-color : 배경색 지정
background-image : 배경 이미지나 그라디언트를 지정한다.
background-position : 배경 이미지의 시작 위치
background-size : 배경 이미지 크기 지정
background-repeat : 배경 이미지 반복 여부 및 방향 결정
background : 위 속성들을 축약한 속성

텍스트 스타일링
font-family : 폰트를 지정
font-size : 폰트의 크기
font-weight : 폰트의 두께
font-style : 폰트 기울기


line-height : 줄 간격
text-align : 텍스트의 정렬
text-indent : 들여쓰기
letter-spacing : 글자 간격
word-spacing : 단어 간격
color : 글씨색 바꾸기
text-decoration : 밑줄,취소선, 윗줄과 같은 장식선
