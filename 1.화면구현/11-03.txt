display : flex

main-axis (주 축) 과
cross-axis (반대 축)이 있다.

만약에 direction : column (위에서 아래로 정렬)
주축 : 세로
반대축 : 가로

컨테이너 속성
flex-direction : 요소들을 어느 방향으로 정렬할건지 주 축을 결정
flex-wrap : 공간이 좁아지면 밑으로 내릴거냐?
justify-content : 주축을 기준으로 요소를 정렬
align-content(요소들이 2줄 이상일 때) : 반대축을 기준으로 요소를 정렬
align-items(요소들이 1줄 이상일 때) : 반대축을 기준으로 요소를 정렬
- stretch : 남은 공간을 늘려서 차지하게 만든다.
- flext-start : 시작지점에 정렬한다.
- flex-end : 마지막지점에 정렬한다.
- center : 중앙정렬한다.
- baseline : 요소들의 글자의 아랫부분에 맞춰서 정렬한다

아이템(자식요소)속성
order : 순서를 정할 수 있다.
flex-grow : 컨테이너 안에서 공간을 얼마나 차지할 것인지 비율을 설정
flex-shrink : 공간이 줄어들 때 얼마나 줄어들 것인지 비율을 설정
flex-basis : 공간을 배분하기 전 기본 너비를 설정
flex : flex-grow, flex-shrink,flex-basis의 축약 속성
align-self : 반대 축을 기준으로 개별 요소의 정렬을 결정
