자바 프로그램 실행 원리

	       컴파일러
코드를 작성 ->  컴파일   ->  JVM이 바이트코드를 읽어서 -> 프로그램 실행
	       (바이트코드로 변환)
	       0,1로 이루어진 컴퓨터가 읽을 수 있는 코드

main이 없으면 자바 프로그램은 실행될 수 없음

ctrl + s : 저장

ctrl + f11 : 실행

주석(comment)
코드에 개발자의 의견이나 설명을 적는 메모
코드에 적더라도 프로그램의 수행에 영향을 끼치지 않는다.
